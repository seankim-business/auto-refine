auto-refine · benchmark-driven bounded optimization

See optimization happen, not just the final result.

This demo shows the same core loop optimizing two different artifact types: a prompt and a workflow policy. Good candidates are kept. Expensive or unsafe ones are rejected. Discarded candidates are rolled back automatically.
